# AliceAndMe - Desktop Companion

A desktop companion Electron app that displays different character moods based on the time of day.
IN PROGRESS

## Getting Started

### Prerequisites

- Node.js (v14+)
- npm

### Installation

```bash
npm install
```

### Running the App

```bash
npm start
```

This launches the Electron app with a borderless, always-on-top window displaying the character.

## Features

- Time-based character mood display (different images for each hour)
- Borderless, always-on-top window
- Hover effects on character image
- Customizable mood images
